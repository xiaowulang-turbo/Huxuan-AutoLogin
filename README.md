# 互选官网自动登录 - 油猴脚本

自动完成 [互选官网](https://huxuan.qq.com/) 的 QQ 密码登录全流程。

## 功能

- 自动点击"广告主登录"
- 自动在 QQ 登录页填充账号密码并登录
- 自动在账户选择页搜索目标账户并点击登录
- 定时检测登录状态，过期自动重新登录
- 支持用户配置 QQ 号、密码、目标账户 ID、检测间隔
- 密码本地加密存储，不上传云端

## 安装

1. 安装 [Tampermonkey](https://www.tampermonkey.net/) 浏览器扩展
2. 点击 Tampermonkey 图标 → 添加新脚本
3. 将 `huxuan-auto-login.user.js` 的内容粘贴进去，保存

## 配置

首次使用时脚本会自动弹出配置窗口，也可以通过以下方式打开：

**Tampermonkey 图标 → 互选官网自动登录 → 设置自动登录**

需要配置：

| 配置项 | 说明 |
|--------|------|
| QQ 号 | 用于登录的 QQ 账号 |
| QQ 密码 | QQ 账号密码 |
| 目标账户 ID | 账户选择页中要登录的账户 ID 数字 |
| 自动检测间隔 | 定时检测登录状态的间隔，单位分钟，默认 30 |
| 启用自动登录 | 开关控制 |

## 使用流程

1. 安装脚本并配置账号信息
2. 访问 `https://huxuan.qq.com/`
3. 脚本自动完成：首页 → QQ 登录 → 账户选择 → 平台首页
4. 登录成功后，脚本定时检测登录状态，过期后自动重新登录

## 注意事项

- 密码存储在浏览器本地（Tampermonkey 存储），不会上传到任何服务器
- 脚本匹配 `huxuan.qq.com`、`sso.e.qq.com`、`*.ptlogin2.qq.com` 三个域名
- 如遇登录验证码等异常情况，需手动处理
